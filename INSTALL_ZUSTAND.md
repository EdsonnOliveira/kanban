# Instalação do Zustand

Para instalar o Zustand, execute um dos seguintes comandos no terminal:

## Opção 1: npm
```bash
npm install zustand
```

## Opção 2: yarn
```bash
yarn add zustand
```

## Opção 3: pnpm
```bash
pnpm add zustand
```

## Opção 4: bun
```bash
bun add zustand
```
